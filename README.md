# hangman-game
This repository contains a Hangman game that was developed for a college work made in Java.

# What you need to run this game
First, ensure you have the javac and java commands avaliable on your system, if don't, you can get these features by install JDK on your computer.
The main class that you need to compile is located in controller/Main.java

# Compiling
From root directory,execute: javac controller/Main.java

# Running
execute the game by running the following command: java controller.Main word1 word2 word3
example:
java controller.Main Elephant Girafe Tiger
you need to pass at least two words as args to run the game, otherwise you will receive a error.
